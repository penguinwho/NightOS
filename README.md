🌙 Night OS

Night OS is a minimal, Arch Linux–based distribution focused on simplicity, speed, and a clean dark aesthetic.
No bloat. No distractions. Just a fast, lightweight system built around control and minimalism.

✨ Philosophy

Night OS follows a simple idea:

Less system. More focus.

Built on the power and flexibility of Arch Linux, Night OS removes unnecessary components and keeps only what matters. It is designed for users who want:

⚡ Maximum performance

🧩 Full control

🌑 A consistent minimal dark theme

🧘 Zero visual clutter

🛠 Base

Night OS is based on:

Arch Linux

Rolling release model

Pacman package manager

AUR support

You get the stability and bleeding-edge updates of Arch, with a curated minimal setup.

🎯 Goals

Keep RAM usage low

Boot fast

Minimal default packages

Clean terminal-first workflow

Optional lightweight window managers

🖥 Default Environment

Night OS ships minimal by design. Depending on the edition:

Tiling WM (Hyprland / i3 edition planned)

No heavy desktop environments by default

Preconfigured dark theme

Clean, distraction-free UI

📦 Included Tools

Only essential tools are preinstalled:

Terminal emulator

File manager (lightweight)

Basic networking tools

Development-ready base

Everything else is up to you.

🚀 Installation (Concept)

Night OS follows an Arch-style installation process:

pacstrap /mnt base linux linux-firmware


From there, Night OS provides:

Minimal configuration scripts

Themed dotfiles

Optional WM setup scripts

🧩 Customization

Night OS is meant to be customized.

You build your workflow.
You choose your tools.
Night OS stays out of your way.

🔥 Why Night OS?

Because minimalism isn’t about removing features —
it’s about removing friction.
